# ThreadWizard Brand Book

## Identity
**Product:** Social media automation for creators  
**Audience:** Content creators, influencers, marketing teams  
**Vibe:** Energetic, multi-platform, creator-focused  

## Color Palette

### Primary Colors
- **Violet:** `#7C3AED` — Primary action, CTAs, core UI
- **Cyan:** `#06B6D4` — Secondary accent, highlights, contrast
- **Black:** `#000000` — Text, backgrounds
- **White:** `#FFFFFF` — Surfaces, text on dark

### Secondary Colors
- **Dark Slate:** `#1E293B` — Backgrounds, cards
- **Neutral Gray:** `#64748B` — Secondary text, borders
- **Success Green:** `#10B981` — Positive actions
- **Warning Orange:** `#F97316` — Alerts, warnings

## Typography

### Font Family
- **Headings:** Inter SemiBold (600)
- **Body:** Inter Regular (400)
- **UI/Labels:** Inter Medium (500)
- **Code:** Fira Code Mono

### Size Scale
- **H1 (Hero):** 48px / 1.2 line-height
- **H2 (Section):** 32px / 1.3 line-height
- **H3 (Subsection):** 24px / 1.4 line-height
- **Body Large:** 16px / 1.5 line-height
- **Body Regular:** 14px / 1.5 line-height
- **Small:** 12px / 1.4 line-height

## Voice & Tone

### Messaging Pillars
1. **Empowering:** Help creators focus on creativity, not logistics
2. **Modern:** Cutting-edge automation, multi-platform support
3. **Direct:** Clear, jargon-free, action-oriented
4. **Energetic:** Enthusiastic without being cheesy

### Tone Guidelines
- ✅ Use active voice: "Schedule posts across 5 platforms"
- ✅ Be specific: "Save 8 hours/week on posting"
- ✅ Speak to creators: "Your time is your most valuable asset"
- ❌ Avoid: Corporate jargon, generic phrases, passive voice

## Visual Style

### Imagery
- Photography: Creators at work, real moment captures
- Illustrations: Modern, flat design, vibrant color use
- Icons: Custom line icons, consistent stroke weight (2px)
- Gradients: Violet → Cyan (horizontal), used sparingly

### Layout Principles
- **Dark-first:** Dark backgrounds default, white/light text
- **High Contrast:** Ensure WCAG AA compliance (4.5:1 minimum)
- **Breathing Room:** Ample whitespace, 16px/24px grids
- **Bold Accents:** Violet CTAs pop against dark backgrounds

### Component Standards
- **Buttons:** Full-width or 140px+, rounded corners (8px), 12px padding vertical
- **Cards:** Dark background (`#1E293B`), subtle border/shadow, gap 16px
- **Forms:** Dark inputs, cyan focus state, error states in orange
- **Navigation:** Minimal, icon-based where possible, violet highlights

## Design System

### Spacing Scale (8px grid)
```
xs: 4px
sm: 8px
md: 16px
lg: 24px
xl: 32px
2xl: 48px
```

### Border Radius
- Buttons/Input: 8px
- Cards: 12px
- Large components: 16px

### Shadows
- **Light:** 0 1px 3px rgba(0,0,0,0.12)
- **Medium:** 0 4px 12px rgba(0,0,0,0.15)
- **Heavy:** 0 12px 32px rgba(0,0,0,0.2)

## Usage Rules

### Do's
✅ Use Violet for primary actions (create, launch, schedule)  
✅ Use Cyan for secondary actions or highlights  
✅ Dark backgrounds with white text always  
✅ Include breathing room around content  
✅ Keep creator focus in copy & design  

### Don'ts
❌ Use light backgrounds (violates dark-first rule)  
❌ Mix multiple gradients in single view  
❌ Reduce contrast below WCAG AA  
❌ Use generic stock photos  
❌ Overly decorative elements (keep minimal)  

## Templates

### Homepage
- Hero: "Schedule once, post everywhere"
- Value props: 3-column grid (time, reach, consistency)
- CTA: Violet button "Start Free"

### Dashboard
- Sidebar: Dark, violet accent on active
- Main content: Grid layout, cards for each platform
- Quick actions: Cyan buttons for common tasks

### Campaign Builder
- Multi-step form: Dark inputs, violet progress indicator
- Preview panel: Live mockup of posts
- Scheduling: Calendar interface, violet selection
